## せどり3ヶ月目の土曜日

リビングのカーペットに段ボール箱が散らばっていた。横にはAmazonの売れ筋ランキングを映したブラウザ、縦には仕入れ値と想定売価を叩き込んだスプレッドシート。せどりを始めて3ヶ月目の土曜日夕方だった。


その週末は近所のリサイクルショップを4軒ハシゴして、Sellers ResearchでAmazon相場を確認しながら100本以上のバーコードを読み取った。バーコードを1本スキャンするたびに0.5秒で相場が出る。その0.5秒を100回繰り返す。差益が出そうなものをカゴに入れて、レジを通して、車で帰宅して、梱包する——全部物理的な作業。腰が痛くなる種類の。


## Excelが正直に吐き出した数字

最初の1ヶ月は宝探しに近い感覚があった。スキャン音が鳴るたびに相場が表示されて、差益ありの商品を見つけた瞬間の「当たり」感はゲームのレアアイテムを引いた時に似ていた。でも3ヶ月目にExcelに真剣に向き合った。


売上合計42,800円。仕入れ原価・送料・Amazon手数料を引いた純利益は9,400円。費やした時間は土日の作業だけで30時間超。時給換算313円——地元の最低賃金の半分以下だった。


「これはビジネスじゃなくて労働だ」。そう気づいた瞬間、続ける理由が消えた。翌朝、Sellers Researchをアンインストールした。


## アフィリエイトに転向、3回詰まって気づいたこと

翌年の春、アフィリエイトに切り替えた。「コンテンツが積み上がれば不労所得になる」という言葉が刺さった。書くこと自体は嫌いじゃなかった。SEOの入門書を2冊買って、WordPressでブログを開設して、週3記事を目標に書き始めた。テーマは「ミニマリストの暮らし」。需要がありそうで競合が少ないと判断した——この判断が甘かった。


3ヶ月後の月間PVは800。アフィリエイト収益はゼロ。書いた記事の9割がGoogle検索100位以下に沈んでいた。週6時間かけて書いたのに「積み上がっている」のは記事の本数だけで、アクセスは微動だにしなかった。


ここでAI自動化に切り替えた。きっかけは「コードが書けなくてもClaude Codeに日本語で指示すればPythonスクリプトが動く」という事実に気づいたことだ。最初は信じなかった。試しに「記事をMarkdownで生成してWordPress APIで投稿するスクリプトを作って」と打ち込んでみた [Claude Code でのスクリプト開発](https://ichinose-taito.com/claude-code-%e5%ae%8c%e5%85%a8%e6%94%bb%e7%95%a5%e3%82%ac%e3%82%a4%e3%83%89-%e3%82%a4%e3%83%b3%e3%82%b9%e3%83%88%e3%83%bc%e3%83%ab%e3%81%8b%e3%82%89%e5%ae%9f%e6%88%a6%e9%81%8b/)。約20分後に動いた。


そこからOllamaをローカルにインストールして qwen2.5:3b を動かし、記事生成・SNS投稿・ [Ollama のモデル選択と日本語性能](https://ichinose-taito.com/ollama-%e3%81%8a%e3%81%99%e3%81%99%e3%82%81%e3%83%a2%e3%83%87%e3%83%ab-10%e9%81%b8%ef%bd%9c%e6%97%a5%e6%9c%ac%e8%aa%9e%e6%80%a7%e8%83%bd%e3%81%ae%e5%ae%9f%e6%b8%ac%e3%83%a9%e3%83%b3%e3%82%ad%e3%83%b3/)KDP出版の3本柱をPythonで一気通貫で回す仕組みを作り始めた。launchd でジョブをスケジュールして、朝7:00・昼12:30・夜20:30に自動投稿が走るようになった。


3回詰まった。最初はWordPress REST APIの認証——アプリケーションパスワードの設定場所を30分以上探した。 [Python による WordPress REST API 実装](https://ichinose-taito.com/python-%e3%81%a7-wordpress-%e8%87%aa%e5%8b%95%e6%8a%95%e7%a8%bf-rest-api-%e5%ae%9f%e8%a3%85%e3%81%a8%e5%ae%9f%e9%81%8b%e7%94%a8%e3%81%ae%e3%83%aa%e3%82%a2%e3%83%ab/)2回目はOllamaの応答が途中で止まる問題で、原因はストリーミングのタイムアウトがデフォルト10秒になっていたこと。3回目はlaunchd のplistファイルのパーミッションで、`chmod 644` しないとジョブが読み込まれない仕様を知らなかった。全部ドキュメントに書いてあることだったけど、コードが読めない人間には「どのドキュメントを読めばいいか」がわからない。Claude Codeにエラーログを貼って「原因を教えて」と聞いたら、ほぼ一発で解決した。


非エンジニアでも「自動ループを回せる」とわかったのが、この1年最大の発見だった。せどりでもアフィリエイトでも、スケールするかどうかは「人間がボトルネックかどうか」で決まる。AIがループを回してくれるなら、人間がやるべきは仕組みを設計することだけでいい——そう腹落ちした瞬間から、作業の質が変わった。
